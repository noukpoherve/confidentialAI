Automatic mode
When on, WARN and ANONYMIZE decisions are handled silently — sensitive data is replaced with placeholders and the prompt is sent automatically with a brief notification. BLOCK decisions always require your attention.

In auto mode, the API's precise server-side redactions are applied — not just local regex. Disable this to always review detections manually via the slide-in panel.

Image Moderation
Scan uploads for sensitive visual content
Enable image moderation
When on, every image you attach or paste is analyzed before it is uploaded. Detected categories include sexual content, graphic violence, hate symbols, self-harm, and harassment. Works on AI platforms and social networks.

Requires an OpenAI API key configured on the backend. Uses the omni-moderation-latest model. If no key is configured the extension fails open — uploads are not blocked.
